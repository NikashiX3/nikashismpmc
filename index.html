Server FAQs

Frequently Asked Questions in Nikashi Network server.

How to register in the lobby?
Its same just like you creating an account the format should like this /register (Password) (RepeatPassword), For example /register ikanasin ikanasin thats mean your account password is ikanasin. And when you try to login just use this command format /login (YourAccountPassword).
How to teleport to Survival World?
You can interact with Random Teleport NPC or using command /rtp.
Where can i check available server commands?
Use type command /help.
How to create a report or reporting a player in the server?
You can use type command /report to open Report GUI, Or you can use type command /reportbug or /reportplayer (PlayerName).
Where can i check server voting for this server?
You can check it at this all one link Platform Portal Nikashi Network in this link : bit.ly/nikashinetwork
Where can i purchase VIP/Crate Key?
You can purchase it in the Discord Server and then go to channel Server Support > Purchase Stuff.
Why does my player data reset to the beginning in Seasonal Event?
Since the Seasonal Event mode is different from SMP, player data from the previous SMP season is restored in the next SMP season. This separation ensures that Seasonal Event data does not affect the main SMP gameplay.
